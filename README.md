# ColoDrone Website

Colorado's Own Drone Service — Denver, CO

## Files
- `index.html` — Main website page
- `css/style.css` — All styles
- `js/main.js` — Interactivity
- `images/` — Add your drone photos and videos here

## Before Going Live — Replace These
- Phone: `(303) 555-0187` → your real number
- Email: `hello@colodrone.com` → your real email
- Address: `Denver, CO 80201` → your real zip
- Stats: Update flight count and other numbers as your business grows

## Adding Your Photos
Place your images in the `/images` folder and reference them in index.html like:
```html
<img src="images/your-photo.jpg" alt="Aerial drone shot Denver Colorado">
```

## Deployment (Vercel + GitHub)
1. Push this folder to a GitHub repo
2. Connect repo to Vercel
3. Add your Porkbun domain in Vercel Settings → Domains
4. Add the DNS records Vercel gives you in Porkbun

## Contact Form
To make the contact form actually send emails, connect it to:
- Formspree (free): https://formspree.io
- Web3Forms (free): https://web3forms.com
